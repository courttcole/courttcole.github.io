HTML tags to remember

TAG NAME DESCRIPTION
<table> table element that starts building a table
<thead> and <tbody> table head and table body groups the header content in a table; groups the body content in a table
<tr> table row defines a new row in a table
<th> and <td> table head and table data defines a header cell in a table; defines a regular data cell in a table
<p> paragraph opening tag for a paragraph
<ul> unordered list defines an unordered, bulleted list
<li> list item used inside ordered lists, unordered lists and menus
<blockquote> blocked quote highlights a quote section, usually indented

CSS properties and values to remember

PROPERTY EXAMPLE VALUES DESCRIPTION
background-color red, rgb (255, 0, 0), hsl (0, 100%, 50%) sets the background color of an element
font-family sans-serif, serif specifies the font of an element
display inline, block, inline-block, flex specifies how an element will display
position static, relative, fixed, absolute, sticky defines position of element, used along with top, bottom, left and right properties

general things to remember

- get familiar with Developer Tools on Chrome
- the Live Server extension in VSCode is great for seeing your code live as you edit
- don't forget to proofread your code for any syntax errors like missing semicolons, closing tags, etc
- there's no one right way to do something but be consistent
